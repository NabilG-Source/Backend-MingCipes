_🍽 MingCipes, this web is a web that provides a food recipe posting system for each user._

## 📍Features

- Login & Register System 🔐
- Multirole: User, Admin 🧑
- Posting, Delete, Update 🔁
- Recipes Category ⛓
- Searching Recipes By Name & Category 🔍
- Dashboard 📊
